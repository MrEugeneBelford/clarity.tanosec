# **App Name**: CyberGuard SME

## Core Features:

- Cybersecurity Assessment: Guided self-assessment questionnaire covering key cybersecurity domains.
- Interactive Feedback: Real-time visual feedback using color-coded responses (green/yellow/red) and a progress bar.
- Weighted Scoring: Weighted scoring system to prioritize critical security areas and generate an overall security score.
- Personalized Recommendations: AI tool that generates specific, actionable recommendations based on assessment responses.
- Priority-Based Action Items: Prioritization of recommendations into high, medium, and low priority action items.
- Professional Reporting: Generates a professional, print-friendly report summarizing the assessment results and recommendations.
- Service CTA: Includes clear call-to-actions (CTAs) for Tanosec services, such as booking a free consultation.

## Style Guidelines:

- Primary color: Tanosec Cybersecurity's primary colour (#bef051) to instill confidence in the tool's reliability.
- Background color: A Dark Grey (#1d1d1b), subtly aligning to the primary color while maintaining a clean interface.
- Accent color: A contrasting orange (#FF9933) to draw attention to CTAs and key information, analogus to the primary color.
- Body and headline font: 'Inter' for a modern, readable, and professional look. Inter is a sans-serif font.
- Use clear and intuitive icons from a consistent set to represent different security domains (e.g., a shield for network security, a lock for access control).
- Implement a clean, mobile-friendly, single-column layout with clearly defined sections for easy navigation and readability.
- Use subtle transitions and progress animations to provide feedback and guide the user through the assessment process.
